\doxysubsection{Descripción detallada}
Clase para publicar mediciones de ozono y temperatura mediante BLE. 

Esta clase permite emitir anuncios BLE con los valores medidos de ozono y temperatura. Se utiliza la clase \doxylink{class_emisora_b_l_e}{Emisora\+BLE} para manejar los anuncios de tipo i\+Beacon.
