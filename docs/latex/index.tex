La emisora BLE está diseñada para transmitir datos de medición a través de un servicio BLE. Este programa utiliza clases para encapsular la funcionalidad de la emisora, las características y el servicio BLE, facilitando su reutilización y mantenimiento.\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Características}{Características}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item Publicación de datos de mediciones (CO2, temperatura, ruido).
\item Interfaz de comunicación por puerto serie.
\item Soporte para múltiples características en un servicio BLE.
\item Posibilidad de añadir características dinámicamente.
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Instalación}{Instalación}}\label{index_autotoc_md4}
