\doxysubsection{Descripción detallada}
Definición de la clase \doxylink{class_publicador}{Publicador} para emitir anuncios BLE de ozono y temperatura. 

Esta clase utiliza una emisora BLE para emitir datos de ozono y temperatura mediante beacons i\+Beacon. Los anuncios se envían con un formato específico que incluye un UUID, major y minor.
